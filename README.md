# NodeJS Magazine Server
인스타그램 형태의 CRUD가 가능한 웹 서버

## 👨‍💻 프로젝트 기간 
- 제작기간 : 2022-02-19 ~ 2022-02-23

___
## Getting Started

```
$ npm -ci
$ npx sequelize db:create
$ npx sequelize db:migrate
$ node app.js
```

## 🔨기술 스택
- NodeJS
- Express
- Sequelize
- JWT(Json Web Token)
- Docker
- Mysql
- AWS EC2
